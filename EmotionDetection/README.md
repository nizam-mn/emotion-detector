# Emotion Detector

This project is an AI-based web application that detects emotions from user input text.

## Features

- Emotion detection logic
- Packaged as Python module
- Unit testing
- Flask web deployment
- Error handling
- Static code analysis

## How to Run

Install dependencies:

pip install -r requirements.txt

Run tests:

python3 test_emotion_detection.py

Run server:

python3 server.py

Open browser:

http://localhost:5000/emotionDetector?textToAnalyze=I am happy
